stage_02 · atoms

The small parts carry the voice

Strip the site down to its smallest pieces and the identity should survive in each one. Every chip is a pill, every hairline is tinted cyan instead of gray, and everything that reads like metadata is set in DM Mono. Hover any specimen. The states are live production CSS.

pills 999px·hairlines cyan .10 / .22· metadata DM Mono·ease .22s quint-out
note / 01

One gradient fill, total

Across the whole site exactly one surface gets a gradient fill: the primary call to action, cyan to green at 90°. Everything else stays a flat token color. Scarcity is what keeps the button worth pressing.

note / 02

Borders are never gray

Every hairline on this page is cyan at 10% or 22% opacity. It reads neutral at a glance, but next to a true gray border the difference is the difference between an instrument panel and a wireframe.

note / 03

Mono means machine

File names, sizes, percentages and status words are set in DM Mono. Sentences stay in DM Sans. The split does quiet work: when a value looks typed by a system, you trust the system exists.

note / 04

States speak in the accent spectrum

The badge set maps the palette to meaning: dim for waiting, cyan for loaded, green for done, purple for the model's verdict, red only for errors. The same mapping holds everywhere on the site, so color alone tells you where a task is.
